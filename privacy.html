Privacy Policy

Last updated: June 23, 2026

What data ListingLens accesses

When you use ListingLens on an Amazon product or review page, it can read publicly visible listing content, visible customer reviews, and review text you manually paste.

ListingLens does not access Amazon login credentials, order history, payment information, Seller Central, or Amazon Ads accounts.

Local storage

Extracted listing data, reviews, settings, AI provider choice, API keys, and generated reports are stored locally in your browser using Chrome storage.

AI analysis

When a Pro user clicks Analyze, selected listing and review content is sent directly from the browser to the chosen AI provider, OpenAI or Google Gemini, using the user's own API key. It does not pass through ListingLens servers.

License and subscription data

If you activate Pro, your purchase email may be sent to the shared Clingogo license verification server to confirm subscription status. Amazon listing and review content is not sent to the license server.

Permissions explained

  • storage — save settings, API keys, cached listing/review data, and reports locally.
  • activeTab / scripting — read listing and visible review content only when you interact with the extension.
  • Amazon host permissions — run on supported Amazon product and review pages.
  • OpenAI / Gemini host permissions — call the selected AI provider directly from the extension.
  • tabs optional and alarms — used only for Advanced Auto Collect after explicit opt-in.

What ListingLens does not do

  • Does not sell data.
  • Does not collect Amazon credentials.
  • Does not access Seller Central.
  • Does not bypass Amazon review limits or CAPTCHA.
  • Does not automate purchases or login.
